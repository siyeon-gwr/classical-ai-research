# Classical AI Research

This repository contains sample code and data for research reproducibility purposes.

## Overview

Two RAG-based classical AI systems built on East Asian historical texts:

- **Zhuxi AI** — Based on *Zhuzi Yulei* (朱子語類), 14,317 utterances
- **Jeongjo AI** — Based on *Hongjae Jeonseo* (弘齋全書) and *Ilseongnok* (日省錄)

## Note on Access

Both systems are currently deployed as commercial services at [askmind.ai](https://askmind.ai).

Full corpus and production code remain proprietary. This repository provides representative samples and pipeline structure for academic reproducibility only.

## Live Demo

- Zhuxi AI: `POST https://askmind.ai/query` 
- Jeongjo AI: `POST https://askmind.ai/query/jeongjo`
