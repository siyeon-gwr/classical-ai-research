# Classical AI Research

This repository contains methodology demonstration code and sample
data for academic reproducibility.

## Overview

Two RAG-based classical AI systems built on East Asian historical
texts:

- **Zhuxi AI** — Based on Zhuzi Yulei (朱子語類), 14,317 utterances
- **Jeongjo AI** — Based on Ildeungrok (日得錄), 2,916 utterances

## Note on Access

Both systems are currently deployed as commercial services at
[askmind.ai](https://askmind.ai). The full corpus, vector database,
production prompt engineering, and runtime infrastructure remain
proprietary and are not distributed.

This repository provides standalone methodology demonstrations and
representative sample data for academic reproducibility only. The
demonstration scripts run independently using the OpenAI Embeddings
API and do not connect to any proprietary infrastructure.
